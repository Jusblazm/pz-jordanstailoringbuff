## What This Mod Does
Provides a context menu option for clothing, which allows players to change tints or textures for clothing that is tintable or has multiple textures, respectively.\
In build 41, you need level 6 tailoring to access the context menu.\
In build 42, there is a sandbox setting that allows you to set the required tailoring level to access the context menu.
